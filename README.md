# Rekomi
ConUHacks X - KAOS Team

### rekomi
Rekomi is a media tracking platform that focuses on human connection instead of algorithms. Built during ConUHacks X, it allows users to manage their movie, anime, and book libraries while sharing direct recommendations with friends.

### The Vision
In a world of AI-driven feeds, Rekomi brings back personal interaction. We built this to be a space where your media library is shaped by the people you trust.

### Tech Stack
Frontend: React, Vite, Tailwind CSS <br>

Backend: Supabase (PostgreSQL) <br>

APIs: TMDb (Movies), Jikan (Anime/Manga), Open Library (Books) <br>

### Features
Unified Media Search: Find movies, anime, and books in one place. <br>

Personal Library: Save items to your "Plan to Watch/Read" lists. <br>

Social Circle: Add friends and view their profiles. <br>

Shared Lists: Create and manage collaborative media lists with your friends.<br>

Direct Recommendations: Send specific media to friends with an optional acceptance system.

Secure Data: Protected by Supabase Row Level Security (RLS).
